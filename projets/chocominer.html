Description du Projet

Site de minage de données réalisé pour LIRMM. Upload de datasets transactionnels avec des statistiques sur le dataset. Ajout de contraintes sur les itemsets recherchés puis retour des résultats avec divers statistiques sur la requête. Front en Vue.js et API back en Java.

Fonctionnalités Principales

  • Upload de dataset
  • Ajout de contrainte sur les itemsets
  • Résolution de la requête
  • API en Java

Technologies Utilisées
